# AI ChatBot

A modern AI chatbot built using Python, Flask, HTML, CSS, JavaScript, SQLite, and Ollama.

## Features

- AI-powered chat
- Chat history
- New Chat option
- SQLite database
- Responsive UI
- Fast and simple interface

## Technologies Used

- Python
- Flask
- HTML
- CSS
- JavaScript
- SQLite
- Ollama

## Installation

```bash
git clone https://github.com/AftabKangani/AI-Chatbot.git
cd AI-Chatbot
pip install -r requirements.txt
python app.py
```
