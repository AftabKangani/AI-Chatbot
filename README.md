# AI ChatBot

A modern AI chatbot built using Python, Flask, HTML, CSS, JavaScript, SQLite, and Ollama.

## Features

- AI-powered chat
- Chat history
- New Chat functionality
- SQLite database
- Responsive user interface
- Fast responses

## Technologies Used

- Python
- Flask
- HTML
- CSS
- JavaScript
- SQLite
- Ollama

## Installation

1. Clone the repository:

```bash
git clone https://github.com/AftabKangani/AI-Chatbot.git
```

2. Open the project:

```bash
cd AI-Chatbot
```

3. Install the required packages:

```bash
pip install -r requirements.txt
```

4. Run the application:

```bash
python app.py
```
